# Executive Summary Dashboard Design

## Objective

Provide a high-level overview of portfolio risk metrics for decision-makers.

## Dashboard Components

### Current VaR

Displays the current Value at Risk of the portfolio.

### Max Drawdown

Shows the maximum observed portfolio decline.

### Portfolio Overview

Summary of portfolio performance and risk exposure.

### Risk Status

Visual indicator of overall portfolio risk level.

## Design Principles

* Simple layout
* Clear KPI visibility
* Minimal clutter
* Easy executive-level interpretation

## Expected Outcome

A dashboard that highlights the most important portfolio risk indicators at a glance.
