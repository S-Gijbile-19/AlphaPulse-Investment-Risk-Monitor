# AlphaPulse Analytics Overview

## Daily Returns

Measures the day-to-day percentage change in stock prices and forms the basis for risk analysis.

## Rolling Volatility

Measures how portfolio risk changes over time using a 30-day rolling window.

## Correlation Analysis

Evaluates relationships between stocks and helps assess portfolio diversification.

## Monte Carlo Simulation

Generates thousands of possible future portfolio outcomes using historical return statistics.

## Value at Risk (VaR)

Estimates the maximum expected portfolio loss at a specified confidence level.

## Statistical Validation

Uses skewness and kurtosis to evaluate whether return distributions are suitable for risk modeling.

## Portfolio Summary

Provides key performance and risk metrics in a consolidated format.
