# Output Files Guide

## rolling_volatility.csv
Contains 30-day rolling volatility values used for risk monitoring.

## correlation_matrix.csv
Shows relationships between portfolio assets for diversification analysis.

## monte_carlo_results.csv
Stores simulation results for future portfolio performance scenarios.

## portfolio_summary.csv
Contains key portfolio performance and risk metrics.

## var_report.csv
Provides Value at Risk estimates for portfolio loss assessment.

## statistical_validation.csv
Contains skewness and kurtosis metrics for return distribution analysis.
