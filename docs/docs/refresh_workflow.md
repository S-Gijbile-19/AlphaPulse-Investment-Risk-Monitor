# Automated Market Data Refresh Workflow

## Workflow Steps

1. Download latest market data from Yahoo Finance.

2. Validate downloaded stock prices.

3. Check for missing values and duplicate records.

4. Store cleaned data in the project data folder.

5. Update portfolio risk calculations.

6. Refresh dashboard visualizations.

7. Verify successful completion of the refresh process.

## Result

The dashboard always uses the latest available market data for risk analysis.
