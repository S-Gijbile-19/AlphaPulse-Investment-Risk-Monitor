# Project Requirements

## Objective

Build a Portfolio Risk Monitoring Dashboard for a boutique investment firm to provide a high-fidelity view of portfolio risk exposure, market uncertainty, and asset diversification.

## Core Features

### Basic Analytics

* Stock Price Trend Analysis
* Daily Returns Analysis
* Portfolio Performance Monitoring
* Interactive Dashboard Filters

### Advanced Analytics

#### Value at Risk (VaR)

Calculate the potential portfolio loss at a selected confidence level to measure downside risk.

#### Monte Carlo Simulation

Perform 10,000 simulations to forecast the future distribution of portfolio performance and generate probability-based risk insights.

#### Correlation Heatmap

Visualize correlations between assets to identify diversification opportunities and concentration risks.

#### Rolling Volatility

Calculate and visualize the 30-day rolling standard deviation of returns to monitor changing market volatility.

## Technology Stack

* Python
* Pandas
* NumPy
* Power BI
* GitHub

## Expected Outcomes

* Real-time portfolio risk monitoring
* Improved diversification analysis
* Enhanced investment decision support
* Interactive risk visualization dashboard
