# Power BI Refresh Configuration Guide

## Objective

Configure Power BI to refresh portfolio data automatically.

## Steps

1. Connect Power BI to the cleaned dataset.
2. Publish the report to Power BI Service.
3. Open Dataset Settings.
4. Configure Scheduled Refresh.
5. Set refresh frequency.
6. Verify successful refresh execution.

## Expected Result

The dashboard updates automatically when new market data becomes available.
