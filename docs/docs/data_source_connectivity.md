# Market Data Source Connectivity

## Data Source

The AlphaPulse project uses Yahoo Finance as the primary market data source.

## Connection Requirements

* Stable internet connection
* Valid ticker symbols
* Access to historical market data
* Consistent date ranges across all assets

## Validation Checks

* Verify data download success
* Check row counts
* Confirm no missing dates
* Validate adjusted closing prices

## Expected Outcome

Reliable market data availability for portfolio risk monitoring and analytics.
